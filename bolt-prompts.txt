СИСТЕМНЫЙ ПРОМПТ — ВСТАВИТЬ ПЕРВЫМ
----------------------------------------

Ты разрабатываешь сайт для компании Рельс-Комплект — оптового поставщика рельсовых материалов (B2B).

СТЕК: чистый HTML5 / CSS3 / Vanilla JavaScript. Никаких React, Vue, Angular — только vanilla JS. Никакого бэкенда — сайт полностью статический.

СТРУКТУРА ФАЙЛОВ:
index.html, catalog.html, product.html, calculator.html, order.html, contacts.html
assets/css/style.css, assets/css/components.css
assets/js/main.js, assets/js/catalog.js, assets/js/calculator.js, assets/js/order.js, assets/js/send.js
data/catalog.json (данные каталога, 158 позиций)

ДИЗАЙН:
Цвет основной: #1A56A0 (синий)
Цвет акцент: #E65100 (оранжевый)
Шрифт: Inter (Google Fonts)
Стиль: строгий промышленный B2B, без лишних украшений
Адаптив: mobile-first, брейкпоинты 768px и 1200px
Ориентир по стилю: сайт vsp74.ru

БИЗНЕС-ЛОГИКА:
- Цены в рублях за тонну
- Позиции без цены → "Цена по запросу"
- Калькулятор: длина(м) × вес_рельса(кг/м) / 1000 = тоннаж
- Заявки: EmailJS (email) + Telegram Bot API (уведомление)
- PDF: jsPDF через CDN

ОГРАНИЧЕНИЯ:
- Без авторизации, корзины, оплаты
- Без платных API
- Комментарии в коде на русском языке

Жди следующего сообщения с конкретной задачей.

ЭТАП 4 — КАЛЬКУЛЯТОР ПУТИ
----------------------------------------

Создай calculator.html — калькулятор пути.
Логику вынести в assets/js/calculator.js

Интерфейс: пошаговый wizard, 3 шага.
Вверху — прогресс-бар с шагами: Рельсы → Шпалы → Результат

ШАГ 1 — Параметры рельсов:
  Тип рельса (select обязательный):
    Р50 (51,67 кг/м), Р65 (64,72 кг/м), Р75 (74,40 кг/м),
    КР70 (70 кг/м), КР80 (80 кг/м), КР100 (100 кг/м), КР120 (120 кг/м)
  Длина пути, м (number, обязательное, min=1)
  Количество нитей (radio): 1 нить / 2 нити (по умолчанию 2)
  Кнопка "Далее →"

ШАГ 2 — Шпалы и скрепления:
  Тип шпал (select):
    Деревянные тип 1 (2 900 руб/шт),
    Деревянные тип 2 (2 400 руб/шт),
    Железобетонные Ш-1 (4 167 руб/шт),
    Железобетонные Ш-3 (4 167 руб/шт),
    Без шпал
  Шаг укладки шпал (select):
    500 мм (2 шт/м), 550 мм (1,82 шт/м), 600 мм (1,67 шт/м)
  Тип скреплений (select):
    Костыльное, Клеммное, Не требуется
  Кнопки "← Назад" и "Рассчитать →"

ШАГ 3 — Результат (генерируется автоматически):
  Таблица спецификации:
  Столбцы: Наименование | Кол-во | Ед.изм | Цена/ед | Сумма
  Строки: рельсы, шпалы (если выбраны)
  Итоговая строка: Общий вес (т) + Ориентировочная стоимость (руб)

  Кнопка "Перенести всё в заявку" → сохранить позиции в localStorage → перейти на order.html
  Кнопка "Скачать PDF-спецификацию" → jsPDF из CDN https://cdnjs.cloudflare.com/ajax/libs/jspdf/2.5.1/jspdf.umd.min.js

ФОРМУЛА расчёта рельсов:
  кол-во рельсов = ceil(длина_пути / 12.5) × кол-во_нитей
  вес_тонн = кол-во_рельсов × 12.5 × вес_кг_на_метр / 1000
  стоимость рельсов = вес_тонн × цена_из_catalog.json (брать минимальную по подкатегории, если null → показать "уточнить")
  кол-во шпал = ceil(длина_пути × шт_на_метр)
  стоимость шпал = кол-во шпал × цена_шпалы

ЭТАП 6 — EMAILJS + TELEGRAM
----------------------------------------

Подключи реальную отправку заявок. Создай assets/js/send.js

В <head> order.html и contacts.html добавь:
<script src="https://cdn.jsdelivr.net/npm/@emailjs/browser@4/dist/email.min.js"></script>

Функция sendOrder(orderData) в send.js:

1. EmailJS:
   emailjs.init("ТВОЙ_PUBLIC_KEY")
   Параметры шаблона:
     to_email: email менеджера
     company: название организации
     contact: контактное лицо
     phone: телефон
     region: регион
     items_table: список позиций (строки через \n)
     total_weight: общий вес в тоннах
     total_price: ориентировочная сумма в руб
     comment: комментарий

2. Telegram (после успешного Email):
   fetch("https://api.telegram.org/bot{BOT_TOKEN}/sendMessage")
   Текст сообщения:
   Новая заявка с сайта
   Компания: название
   Контакт: имя, телефон
   Регион: регион
   Позиции: список через запятую
   Итого: ~X руб

3. UX во время отправки:
   - Кнопка "Отправить заявку" → disabled + текст "Отправляем..."
   - Успех → зелёный блок "Заявка отправлена! Мы свяжемся с вами." + очистить localStorage позиций
   - Ошибка → красный блок "Ошибка отправки. Позвоните нам: [телефон]" + кнопка снова активна

Замени alert в order.html на вызов sendOrder().

ВАЖНО: ключи EmailJS и Telegram хранить как константы в начале send.js
